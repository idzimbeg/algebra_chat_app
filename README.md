# About Algebra Chat App

This project was created as a final exam for frontend web-developer course at University of Algebra.
-App is initialized using npx create-react-app with typescript.
-Base configuration for this project was created depending on the eslint rules, husky for pre-commit hooks and better control of the commits that are trying to be merged and prettier for final file/code structure-including imports and exports.
-UI styling is done using tailwindcss and daisyUI plugin.
-For state management react-query and react-hook-form are used.
-Backend is done using firebase and firestore.

## Starting With Chat App

Clone this repository to your local folder and run ### `npm start` to install all dependencies.
